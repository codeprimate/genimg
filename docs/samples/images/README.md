# Sample Images

This directory contains sample reference images for testing and examples.

## Images

- `reference_style.jpg` - Example of a stylistic reference image
- `reference_subject.png` - Example of a subject reference image

## Usage

These images can be used with genimg for testing reference-based generation:

```bash
genimg generate "same subject in watercolor style" --reference reference_style.jpg
```
